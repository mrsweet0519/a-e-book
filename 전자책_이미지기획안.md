# 전자책 분량 확장 및 시각화(이미지 삽입) 기획안

기존 51페이지 분량의 텍스트 원고에 실제 화면 캡처, 도식, 예시 카드 등을 삽입하고, 각 장의 시작과 체크리스트를 단독 페이지로 분리하여 **90~110페이지 분량의 '실전형/시각화 특화 전자책'**으로 확장하기 위한 가이드입니다.

---

## 1. 개인정보 보호 및 캡처 가이드라인 (필수)
실제 화면을 캡처할 때, 아래 정보는 캔바/미리캔버스에서 '블러(흐림) 처리' 하거나 불투명한 '검은색/회색 네모 박스'로 가려야 합니다.
- **유튜브 스튜디오 캡처 시:** 우측 상단의 내 프로필 사진, 채널 이름, 이메일 주소, 민감한 실제 수익 금액(공개 원치 않을 시)
- **AI 툴 / Pexels 캡처 시:** 로그인된 내 계정 정보, 결제 카드 정보 등
- **주의:** 저작권 문제가 발생할 수 있는 '타인의 유튜브 영상 썸네일'을 캡처할 때는 얼굴 부분을 블러 처리하거나 채널명을 가리는 것이 안전합니다.

## 2. 장별 이미지 및 시각 자료 삽입 리스트 (총 32개 제안)

1. **유튜브 플레이리스트 채널 성공 예시**
   - 종류: 실제 화면 캡처
   - 내용: 유튜브 검색창에 '수면 음악'을 쳤을 때 나오는 조회수 높은 플레이리스트 목록 캡처
   - 캡션: "우리에게 친숙한 수백만 조회수의 플레이리스트 영상들"

2. **수익화 파이프라인 개념도**
   - 종류: 도식 (디자인 카드)
   - 내용: 계단식으로 우상향하는 그래프 아이콘 + "꾸준한 업로드 ➡️ 알고리즘 노출 ➡️ 시청 시간 누적 ➡️ 파이프라인 완성" 텍스트
   - 캡션: "음악 채널은 단기 대박이 아닌 누적형 파이프라인입니다."
   - 제작: 캔바/미리캔버스 도형 활용

## 3. 페이지 레이아웃 확장 기준 (어떻게 100페이지가 되는가?)

페이지를 억지로 늘리지 않아도, '독자 친화적'으로 여백과 시각 자료를 주면 자연스럽게 90~110페이지가 완성됩니다.

1. **장(Chapter) 타이틀 단독 페이지 (약 11쪽 추가)**
   - 1장~9장 및 프롤로그/에필로그가 시작될 때마다 **"제목 + 💡핵심요약"만 들어간 큰 대문 페이지**를 1장씩 삽입합니다.
2. **할 일 & 체크리스트 단독 페이지 (약 10쪽 추가)**
   - 매 장 끝에 있는 `[🚀 초보자가 바로 해야 할 일]`과 `[✅ 체크리스트]`를 이전 본문과 분리하여, 새로운 페이지 1장에 큼직하게 배치합니다. (워크북 느낌 부여)
3. **이미지 삽입에 따른 본문 밀림 (약 20~25쪽 추가)**
   - 위에서 기획한 32개의 이미지가 본문에 중간중간 들어가면 (이미지 크기는 반 페이지~한 페이지 차지), 자연스럽게 텍스트가 뒤로 밀리며 약 20쪽 이상의 분량이 늘어납니다.
4. **부록 템플릿 1페이지 1템플릿 원칙 (약 10쪽 추가)**
   - 부록 10종을 줄글로 붙여놓지 않고, '표지 ➡️ 채널 기획표(1쪽) ➡️ 타깃 분석표(1쪽)' 식으로 단독 페이지화 합니다.

**[최종 예상 페이지 계산]**
- 기존 텍스트 원고: 51쪽
- 장 표지 및 체크리스트 분리: +21쪽
- 이미지 32종 삽입 여백: +20쪽
- 부록 단독 페이지화: +5쪽
- **총 예상 분량: 약 97 ~ 105쪽 (100페이지 내외 최적의 실무서 완성)**

## 4. 🌟 크몽 미리보기 이미지 추천 (5~7장)

크몽 상품 등록 시 상세페이지 상단에 들어가는 미리보기 이미지(워터마크 필수)로 쓰기 가장 좋은, 독자의 구매 욕구를 자극하는 '시각 자료가 포함된 페이지' 6가지를 추천합니다.

1. **[표지]** 메인 표지 (얼굴 없는 유튜브 음악 채널 수익화 가이드)
