# 실제 화면 캡처 체크리스트 (작업 순서별 정리)

전자책 디자인 작업 시 작성자가 직접 PC에서 접속하여 캡처해야 할 화면들의 목록입니다. 효율적인 캡처 작업을 위해 사이트 및 프로그램 종류별로 묶어 두었습니다.

---

## 1. 📺 유튜브 관련 화면 캡처

**[1] 유튜브 플레이리스트 채널 성공 예시**
- **접속 사이트/메뉴:** 유튜브 메인 검색창
- **캡처할 부분:** '수면 음악' 검색 결과에 나오는 수백만 조회수의 플레이리스트 영상 목록 (썸네일+조회수)
- **가릴 개인정보:** 타 채널의 채널명 및 프로필 사진 (모자이크 또는 블러)
- **캡션 문구:** "우리에게 친숙한 수백만 조회수의 플레이리스트 영상들"
- **전자책 위치:** 프롤로그

**[2] YPP 수익화 조건 (1,000명 / 4,000시간)**
- **접속 사이트/메뉴:** 유튜브 스튜디오 ➡️ 좌측 [수익 창출] 메뉴
- **캡처할 부분:** 구독자 1,000명 / 시청 시간 4,000시간 달성 게이지 바 부분
- **가릴 개인정보:** 우측 상단 내 프로필 사진 및 계정 이름, 민감한 채널 데이터
- **캡션 문구:** "2026년 기준 유튜브 수익 창출 기본 요건"
- **전자책 위치:** 2장. 2026년 수익화 기준과 주의사항

**[3] 잘 쓴 채널 소개글 예시**
- **접속 사이트/메뉴:** 모범이 될 만한 유튜브 타 채널 접속 ➡️ [정보] 또는 [채널 소개] 탭
- **캡처할 부분:** 키워드가 자연스럽게 녹아든 채널 소개글 텍스트 영역
- **가릴 개인정보:** 타 채널명 부분 (블러 처리)
- **캡션 문구:** "알고리즘과 시청자 모두를 만족시키는 채널 소개글 예시"
- **전자책 위치:** 3장. 채널 콘셉트 기획하기

**[4] 업로드 제목/설명란 작성 화면**
- **접속 사이트/메뉴:** 유튜브 스튜디오 ➡️ 우측 상단 [만들기] ➡️ 영상 업로드 [세부정보] 창
- **캡처할 부분:** 제목 입력칸과 타임스탬프가 들어간 설명란 입력칸
- **가릴 개인정보:** 우측에 뜨는 내 동영상 미리보기 이미지(썸네일), 채널 프로필
- **캡션 문구:** "키워드가 포함된 제목과 타임스탬프 세팅 예시"
- **전자책 위치:** 6장. 유튜브 업로드 최적화

**[5] 유튜브 스튜디오 태그(Tag) 입력란**
- **접속 사이트/메뉴:** 영상 업로드 [세부정보] 창 하단의 [자세히 보기] 클릭 ➡️ [태그] 영역
- **캡처할 부분:** 쉼표로 구분하여 입력된 태그들이 파란 박스로 생성되어 있는 영역
- **가릴 개인정보:** 없음
- **캡션 문구:** "설명란의 해시태그(#)와 하단 태그(Tag)는 다릅니다."
- **전자책 위치:** 6장. 유튜브 업로드 최적화

**[6] 저작권 '발견된 문제 없음' 화면 (★중요)**
- **접속 사이트/메뉴:** 영상 업로드 창의 3단계 [검토] 탭
- **캡처할 부분:** '저작권: 발견된 문제 없음' 텍스트와 초록색 체크마크(✔️) 부분
- **가릴 개인정보:** 우측 영상 미리보기 및 링크
- **캡션 문구:** "업로드 전 반드시 눈으로 확인해야 할 저작권 검토 통과 마크"
- **전자책 위치:** 6장. 유튜브 업로드 최적화

**[7] 숏츠 '관련 동영상' 링크 연결 화면**
- **접속 사이트/메뉴:** 유튜브 스튜디오 ➡️ 숏츠 영상 [세부정보] 창 우측 메뉴
- **캡처할 부분:** '관련 동영상' 항목을 클릭하여 내 1시간짜리 롱폼 영상을 선택하는 팝업 화면
- **가릴 개인정보:** 연결하는 영상 외의 개인적인 채널 영상 목록
- **캡션 문구:** "숏츠 조회수를 롱폼 시청 시간으로 연결하는 필수 세팅"
- **전자책 위치:** 7장. 숏츠 확장 전략

## 2. 🎵 AI 음원 관련 화면 캡처

**[8] Soundraw 요금제 및 라이선스 확인**
- **접속 사이트/메뉴:** Soundraw(사운드로우) 홈페이지 ➡️ [Pricing] 또는 [License] 메뉴
- **캡처할 부분:** Creator 요금제표와 그 아래 'Commercial use(상업적 이용)' 또는 'YouTube monetization(유튜브 수익창출)'이 허가된다고 적힌 문구
- **가릴 개인정보:** 없음 (단, 로그인되어 있다면 우측 상단 이메일/프로필 가림)
- **캡션 문구:** "Soundraw 유료 플랜의 유튜브 수익 창출 허가 규정"
- **전자책 위치:** 4장. AI음원 제작하기

**[9] Soundraw 곡 세팅 화면**
- **접속 사이트/메뉴:** Soundraw [Create Music] 버튼 클릭 후 화면
- **캡처할 부분:** Genre(장르), Mood(무드), Tempo(템포) 등을 선택하는 동그란 버튼들이 나열된 상단 바 영역
- **가릴 개인정보:** 없음
- **캡션 문구:** "클릭 세 번만으로 채널 분위기에 맞는 곡이 생성됩니다."
- **전자책 위치:** 4장. AI음원 제작하기

## 3. 🎬 무료 영상 소스 관련 화면 캡처

**[10] Pexels 검색 팁 및 라이선스 확인**
- **접속 사이트/메뉴:** Pexels(픽셀즈) 홈페이지 메인 및 영상 상세 페이지
- **캡처할 부분:** 
  1. 상단 검색창에 영어(예: Night sky)로 검색하고 '동영상(Videos)' 탭을 누른 화면
  2. 영상 우측 [정보/라이선스] 탭의 '상업적 이용 가능(Free to use)' 문구 (빨간 네모 박스 처리 추천)
- **가릴 개인정보:** 우측 상단 로그인 프로필 아이콘
- **캡션 문구:** "한글보다는 영어로 검색해야 양질의 소스를 얻습니다. / 무료 소스라도 상업적 이용 가능 여부는 반드시 직접 확인해야 합니다."
- **전자책 위치:** 2장. 수익화 기준(라이선스), 5장. 영상 소스와 편집하기

## 4. ✂️ 편집 프로그램 관련 화면 캡처 (CapCut, Filmora 등)

**[11] 영상 편집 프로그램 기본 인터페이스**
- **접속 사이트/메뉴:** PC용 CapCut(캡컷) 또는 Filmora(필모라) 실행 화면
- **캡처할 부분:** 화면 하단 타임라인 영역 (위쪽에는 비디오 클립, 아래쪽에는 오디오 파일이 배치된 전체적인 조립 구조)
- **가릴 개인정보:** 내 PC의 파일 경로명이나 개인적인 폴더 이름이 보일 경우 모자이크
- **캡션 문구:** "초보자도 직관적으로 다룰 수 있는 편집 툴 타임라인"
- **전자책 위치:** 5장. 영상 소스와 편집하기

**[12] 트랜지션(화면 전환) 적용 화면**
- **접속 사이트/메뉴:** 편집 프로그램 내 타임라인
- **캡처할 부분:** 비디오 클립과 클립 사이에 '디졸브(Dissolve)' 등의 화면 전환 효과 아이콘이 삽입된 부분을 크게 줌인(확대)한 화면
- **가릴 개인정보:** 없음
- **캡션 문구:** "반복 콘텐츠를 피하기 위한 필수 조미료, 화면 전환(디졸브)"
- **전자책 위치:** 5장. 영상 소스와 편집하기
