# HTML/CSS 전자책 PDF 제작 안내

이 폴더는 `유튜브음악채널_수익화가이드_2026_최종원고.md` 원고를 A4 세로 PDF 출력용 HTML/CSS 구조로 변환한 결과입니다.

## 파일 구성

- `index.html`: 표지, 저작권 안내, 목차, 장 시작 페이지, 본문, 부록, 마무리 페이지가 들어간 전자책 본문
- `style.css`: A4 세로 출력, 표지, 색상, 박스 디자인, 표, 이미지 자리, 인쇄용 페이지 나누기 스타일
- `assets/cover.png`: 첫 페이지 전체에 들어가는 표지 이미지
- `make-pdf.mjs`: Playwright로 최종 PDF를 생성하는 스크립트

## 최종 PDF 생성 명령어

프로젝트 루트에서 아래 명령을 실행합니다.

```powershell
node ebook/make-pdf.mjs
```

생성 파일은 아래 위치에 저장됩니다.

```text
ebook/youtube_music_channel_guide_2026.pdf
```

`make-pdf.mjs`는 내부에서 Playwright CLI를 호출합니다. 처음 실행할 때 Playwright가 설치되어 있지 않으면 `npx`가 필요한 파일을 내려받을 수 있습니다.

## 생성 후 확인 방법

1. `ebook/youtube_music_channel_guide_2026.pdf`를 엽니다.
2. 1쪽 표지가 A4 한 페이지를 꽉 채우고, 푸터와 페이지 번호가 없는지 확인합니다.
3. 2쪽부터 하단 푸터와 페이지 번호가 본문과 겹치지 않는지 확인합니다.
4. 각 장 시작 페이지가 새 페이지에서 시작되는지 확인합니다.
5. 표, 체크리스트, 주의 박스, 실전 예시 박스가 페이지 중간에서 어색하게 잘리지 않는지 확인합니다.
6. 이미지 자리 32개가 실제 이미지 없이도 깔끔한 시각 자료 영역처럼 보이는지 확인합니다.

## 브라우저에서 PDF로 저장하는 방법

Playwright 생성 방식을 권장합니다. 브라우저에서 직접 저장해야 한다면 아래 설정을 사용합니다.

1. `ebook/index.html` 파일을 Chrome 또는 Edge로 엽니다.
2. `Ctrl + P`를 누릅니다.
3. 대상에서 `PDF로 저장`을 선택합니다.
4. 용지는 `A4`, 방향은 `세로`로 설정합니다.
5. `배경 그래픽`을 켭니다.
6. 브라우저 자체의 머리글/바닥글 옵션은 끕니다.
7. 미리보기에서 표지와 하단 푸터 위치를 확인한 뒤 저장합니다.

## 이미지 자리 사용 기준

본문에는 이미지 자리 32개가 있습니다. 실제 판매본에서는 화면 캡처, 도식, 표 이미지로 교체하면 좋습니다. 이미지가 준비되지 않은 상태에서도 최종본 느낌을 해치지 않도록 placeholder는 `시각 자료` 형태로 정리되어 있습니다.

- 우선 교체 권장: YPP 조건, 재사용/반복 콘텐츠 거절 화면, 상업적 이용 약관, AI 음원 요금제/세팅, Pexels 검색, 편집 타임라인, 업로드/저작권 검토, 숏츠 연결 화면
- 없어도 진행 가능: 장르 무드보드, 장점 요약, 주차별 플랜 표 미리보기, 부록 템플릿 미리보기 이미지

## 편집 시 주의사항

- 본문 문구를 줄이거나 삭제하지 말고, 필요한 경우 이미지와 여백만 조정합니다.
- 과장된 성과, 수익 보장, 심사 승인 확정 표현은 추가하지 않습니다.
- 캡처 이미지에는 채널명, 이메일, 결제 정보, 민감한 수익 정보가 보이지 않도록 가림 처리합니다.
- 최종 판매용 PDF를 만들기 전에는 전체 페이지를 한 번 훑어 표와 박스가 잘리지 않는지 확인합니다.
